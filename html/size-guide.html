HƯỚNG DẪN CHỌN SIZE GIÀY

Cách đo size chân

Để chọn được đôi giày phù hợp, bạn cần đo chính xác kích thước bàn chân của mình. Dưới đây là hướng dẫn chi tiết:

Bước 1: Chuẩn bị

  • Một tờ giấy trắng (khổ A4)
  • Bút chì hoặc bút bi
  • Thước đo

Bước 2: Đo kích thước

  • Đặt tờ giấy trên sàn phẳng, dựa vào tường
  • Đứng thẳng trên tờ giấy, gót chân chạm tường
  • Dùng bút đánh dấu điểm xa nhất của ngón chân
  • Đo khoảng cách từ cạnh giấy (phía tường) đến dấu bút
Lưu ý: Nên đo vào cuối ngày khi chân đã giãn ra hoàn toàn. Nếu hai chân có kích thước khác nhau, hãy chọn size theo chân lớn hơn.

Hướng dẫn chọn size theo từng thương hiệu

NIKE

Lưu ý: Giày Nike thường có form hơi nhỏ, nên chọn tăng 0.5 size so với bình thường.

Các dòng giày khác nhau có thể có form khác nhau:

  • Air Force 1: True to size hoặc down 0.5 size
  • Air Jordan 1: True to size
  • Air Max: True to size hoặc up 0.5 size

ADIDAS

Lưu ý: Giày Adidas thường có form chuẩn, nên chọn đúng size chân.

Các dòng giày khác nhau có thể có form khác nhau:

  • Ultraboost: True to size hoặc up 0.5 size
  • Stan Smith: True to size
  • NMD: Down 0.5 size

JORDAN

Lưu ý: Giày Jordan thường có form chuẩn, nên chọn đúng size chân.

Các dòng giày khác nhau có thể có form khác nhau:

  • Air Jordan 1: True to size
  • Air Jordan 4: True to size
  • Air Jordan 11: True to size hoặc up 0.5 size

YEEZY

Lưu ý: Giày Yeezy thường có form nhỏ, nên chọn tăng 0.5-1 size so với bình thường.

Các dòng giày khác nhau có thể có form khác nhau:

  • Yeezy 350 V2: Up 0.5 size
  • Yeezy 500: True to size
  • Yeezy 700: True to size hoặc up 0.5 size

Mẹo chọn size giày phù hợp

Dấu hiệu giày vừa vặn

  • Khoảng trống 0.5-1cm từ ngón chân dài nhất đến mũi giày
  • Không bị chật ở phần mu bàn chân
  • Gót chân không bị xê dịch khi di chuyển
  • Không cảm thấy đau hoặc khó chịu khi đi

Dấu hiệu giày quá nhỏ

  • Ngón chân chạm vào mũi giày
  • Cảm thấy tê hoặc đau ở ngón chân
  • Khó khăn khi xỏ chân vào giày
  • Phần mu bàn chân bị ép chặt

Dấu hiệu giày quá rộng

  • Gót chân bị xê dịch nhiều khi đi
  • Khoảng trống quá lớn ở mũi giày
  • Bàn chân trượt về phía trước khi đi
  • Dễ bị phồng rộp do ma sát

Thời điểm tốt nhất để đo chân

  • Vào cuối ngày khi chân đã giãn ra hoàn toàn
  • Đo cả hai chân và chọn size theo chân lớn hơn
  • Đo khi đang đứng, không ngồi
  • Đo khi đang mang tất mà bạn thường dùng với giày
Lưu ý quan trọng: Mỗi thương hiệu và mỗi dòng giày có thể có form khác nhau. Bảng size chỉ mang tính tham khảo. Nếu có thể, hãy thử giày trực tiếp trước khi mua.
